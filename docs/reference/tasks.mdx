## Overview

Tasks represent specific activities performed during a visit. Certain programs **require** tasks to be documented, while others make them optional.

<Warning>
  **Required for**: ABP, AUP, CHP, PCP, MHP programs
</Warning>

## Task Requirements by Program

| Program | Tasks Required | Task ID Range |
|---------|----------------|---------------|
| **HHI** (Home Health Non-Waiver) | Optional | 130-149, 801-816 |
| **ABI, AUI, CHI, PCI** (Waiver) | Optional | 130-149, 801-816 |
| **ABP** (ABI Non-Clinical) | **Required** | 1-99, 150-158 |
| **AUP** (Autism Professional) | **Required** | 99, 160-173 |
| **CHP** (CT Home Care) | **Required** | 1-99, 150-158 |
| **PCP** (Personal Care Assistant) | **Required** | 1-99, 150-158 |
| **MHP** (Mental Health) | **Required** | 97, 99, 201-225 |

## Usage in API

### Single Task

```json
{
  "Tasks": [{
    "TaskID": "130",
    "TaskReading": null,
    "TaskRefused": false
  }]
}
```

### Multiple Tasks

```json
{
  "Tasks": [
    {
      "TaskID": "130",
      "TaskReading": null,
      "TaskRefused": false
    },
    {
      "TaskID": "136",
      "TaskReading": null,
      "TaskRefused": false
    },
    {
      "TaskID": "146",
      "TaskReading": null,
      "TaskRefused": false
    }
  ]
}
```

### Task with Reading

```json
{
  "Tasks": [{
    "TaskID": "802",
    "TaskReading": "98.6",
    "TaskRefused": false
  }]
}
```

### Refused Task

```json
{
  "Tasks": [{
    "TaskID": "130",
    "TaskReading": null,
    "TaskRefused": true
  }]
}
```

## Validation Rules

<Warning>
  **Task Validation Rules**
</Warning>

### Required Programs

Tasks **must** be provided for:
- ABP (Acquired Brain Injury Non-Clinical)
- AUP (Autism Professional)
- CHP (CT Home Care Program)
- PCP (Personal Care Assistant)
- MHP (Mental Health Program)

### Task ID Validation

- TaskID must match program's task list
- TaskID must be a valid string (not integer)
- Multiple tasks can be provided per visit

### Task Reading

- Optional for most tasks
- Required for certain clinical tasks (e.g., vital signs)
- Max 10 characters
- No special characters

### Task Refused

- Boolean: `true` or `false`
- Indicates client refused the task
- Task still counts as attempted

## Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| Missing tasks | Tasks required but not provided | Add Tasks segment |
| Invalid TaskID | TaskID doesn't match program | Use valid TaskID from program list |
| Wrong task range | Using HH tasks for PCP program | Use correct task range for program |
