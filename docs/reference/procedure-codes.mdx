## Overview

Procedure codes (also called Revenue Center Codes or HCPCS codes) identify the specific services provided during a visit. Each payer/program combination has its own set of valid codes.

<Info>
  **Note**: Codes must match exactly as shown, including modifiers in the correct order.
</Info>

## Modifier Usage

### Common Modifiers

| Modifier | Meaning |
|----------|---------|
| **TT** | Subsequent Client |
| **U2** | One Time Only (1X Only) |
| **TU** | Overtime |
| **TG** | Complex Care |
| **TH** | Obstetrics |
| **TE** | LPN Modifier |
| **TD** | Initial Assessment |
| **GW** | Not Applicable to Hospice |

### Modifier Rules

<Warning>
  **Important**: Modifiers must be in the correct order
</Warning>

1. Modifiers must be applied in order (Modifier1, then Modifier2, etc.)
2. Cannot skip modifier positions
3. Use `"NULL"` for unused modifiers
4. Maximum 4 modifiers per code

**Example**:
```json
{
  "ProcedureCode": "S9123",
  "Modifier1": "TG",
  "Modifier2": "TT",
  "Modifier3": "NULL",
  "Modifier4": "NULL"
}
```
