## Validation Rules

<Warning>
  **Important Validation Rules**
</Warning>

### Valid Combinations Only

The system will reject invalid payer/program combinations:

✅ **Valid**:
- `CTHH` + `HHI`
- `CTDSS` + `ABI`, `AUI`, `CHI`, `PCI`, `ABP`, `AUP`, `CHP`, `PCP`
- `CTMHW` + `MHP`

❌ **Invalid**:
- `CTHH` + `ABP` (wrong payer)
- `CTMHW` + `HHI` (wrong payer)
- `CTDSS` + `MHP` (wrong payer)

### Task Requirements

**If PayerProgram is ABP, AUP, CHP, PCP, or MHP**:
- Tasks segment is **REQUIRED**
- At least one valid TaskID must be provided
- TaskID must match program's task list

**If PayerProgram is HHI, ABI, AUI, CHI, or PCI**:
- Tasks are optional
- If provided, must be valid task IDs

### Procedure Code Validation

Each payer/program combination has specific valid procedure codes:

```json
// Valid for CTHH/HHI
{
  "PayerID": "CTHH",
  "PayerProgram": "HHI",
  "ProcedureCode": "T1004"  // Valid
}

// Invalid combination
{
  "PayerID": "CTHH",
  "PayerProgram": "HHI",
  "ProcedureCode": "1021Z"  // Invalid - this is for CTDSS/ABP
}
```

## Program-Specific Requirements

### Home Health Programs (HHI, ABI, AUI, CHI, PCI)

**Service Focus**: Medical and therapeutic services
- Nursing care (RN, LPN)
- Physical therapy
- Occupational therapy
- Speech therapy
- Home health aide services

**Documentation**: Standard home health documentation

### Non-Clinical Programs (ABP, AUP, CHP, PCP)

**Service Focus**: Personal care and support services
- Personal care assistance
- Companion services
- Homemaker services
- Respite care
- Community living support

**Documentation**: Task completion required

### Mental Health Program (MHP)

**Service Focus**: Mental health and recovery services
- Recovery assistance
- Community support
- Peer services
- Mental health counseling
- Case management

**Documentation**: Task completion and progress notes required

## Common Errors

| Error | Cause | Solution |
|-------|-------|----------|
| Invalid payer/program | Wrong combination | Use valid combinations from matrix |
| Missing tasks | Tasks required but not provided | Add Tasks segment for ABP/AUP/CHP/PCP/MHP |
| Invalid procedure code | Code doesn't match program | Use codes from correct appendix |
| Wrong payer ID | Typo or incorrect payer | Use exact values: CTDSS, CTHH, or CTMHW |
