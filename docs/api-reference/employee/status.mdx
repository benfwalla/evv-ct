## Overview

After submitting employee data via POST, use this endpoint to check if the submission has been processed successfully or if there were any errors.

<Info>
  **Wait Time**: Wait at least 5 minutes after POST before checking status. The UUID is available for 48 hours.
</Info>

## Endpoint

<CodeGroup>

```bash Production
GET https://api.sandata.com/interfaces/intake/employees/rest/api/v1.1/status?uuid={uuid}
```

```bash UAT
GET https://uat-api.sandata.com/interfaces/intake/employees/rest/api/v1.1/status?uuid={uuid}
```

</CodeGroup>

<ParamField header="Authorization" type="string" required>
  Basic Authentication header
  
  Format: `Basic <base64(username:password)>`
</ParamField>

<ParamField header="Account" type="string" required>
  Your Sandata account ID (or use `EntityGuid` for multi-account)
</ParamField>

## Query Parameters

<ParamField query="uuid" type="string" required>
  The UUID returned from the POST request
  
  **Example**: `7f6dcd1a-ec5e-4efd-a2d4-1049756016a5`
</ParamField>

```json
  "status": "SUCCESS",

  "data": {
```

## Example Requests

<CodeGroup>

```bash cURL
curl -X GET "https://api.sandata.com/interfaces/intake/employees/rest/api/v1.1/status?uuid=7f6dcd1a-ec5e-4efd-a2d4-1049756016a5" \
```

```javascript JavaScript
const uuid = '7f6dcd1a-ec5e-4efd-a2d4-1049756016a5';

const checkStatus = async () => {
  const response = await fetch(
    `https://api.sandata.com/interfaces/intake/employees/rest/api/v1.1/status?uuid=${uuid}`,
    {
      headers: {
        'Authorization': 'Basic dXNlckBleGFtcGxlLmNvbTpzZWNyZXQ=',
        'Account': '12345'
      }
    }
  );
  
  const result = await response.json();
  
  if (result.messageSummary === "All records updated successfully.") {
    console.log('✅ All employees processed successfully');
    return true;
  } else if (result.messageSummary.includes("not ready yet")) {
    console.log('⏳ Still processing, check again in 5 minutes');
    return false;
  } else {
    console.log('❌ Errors found:', result.data);
    return false;
  }
};

// Wait 5 minutes before first check
setTimeout(checkStatus, 5 * 60 * 1000);
```

```python Python
import requests

        params={'uuid': uuid},
        headers={
            'Authorization': 'Basic dXNlckBleGFtcGxlLmNvbTpzZWNyZXQ=',
            'Account': '12345'

    )
    
    result = response.json()
    
    if result['messageSummary'] == "All records updated successfully.":
        print('✅ All employees processed successfully')
        return True
    elif "not ready yet" in result['messageSummary']:
        print('⏳ Still processing, check again in 5 minutes')
        return False
    else:
        print('❌ Errors found:', result['data'])
        return False

# Wait 5 minutes before first check
uuid = '7f6dcd1a-ec5e-4efd-a2d4-1049756016a5'
time.sleep(300)
check_employee_status(uuid)
```

## Common Error Codes

| Error Code | Meaning | Solution |
|------------|---------|----------|
| `-100` | Missing required field | Add the required field (SSN, name, etc.) |
| `-101` | Invalid field format | Fix format (email, SSN, etc.) |
| `-103` | Invalid characters | Remove special characters from name |
| `-709` | Duplicate SequenceID | Increment SequenceID and resubmit |
| `-710` | Employee not found | Verify EmployeeIdentifier |

## Polling Strategy

```javascript
async function pollEmployeeStatus(uuid, maxAttempts = 10) {
  for (let attempt = 1; attempt <= maxAttempts; attempt++) {
    console.log(`Checking employee status (${attempt}/${maxAttempts})...`);
    
    const response = await fetch(
      `https://api.sandata.com/interfaces/intake/employees/rest/api/v1.1/status?uuid=${uuid}`,
      {
        headers: {
          'Authorization': 'Basic dXNlckBleGFtcGxlLmNvbTpzZWNyZXQ=',
          'Account': '12345'
        }
      }
    );
    
    const result = await response.json();
    
    // Check if done processing
    if (!result.messageSummary.includes("not ready yet")) {
      return result;
    }
    
    // Wait 5 minutes before next check
    if (attempt < maxAttempts) {
      await new Promise(resolve => setTimeout(resolve, 5 * 60 * 1000));
    }
  }
  
  throw new Error('Status check timed out');
}
```
